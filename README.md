## Para testar em sua maquina

tenha o NodeJS >= 18
```bash

# Clone no seu repositório
$ git clone git@github.com:lipecalixto/fundamentos-tailwind-ignite.git

# Acesse a o diretório
$ cd fundamentos-tailwind-ignite

# Instale às Dependências
$ pnpm install

# Rode a aplicação
$ pnpm dev

```
